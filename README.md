# CHIOMA LP — 3 Design Studies

CHIOMA（コラティーナ種100% EVOO）LPの3案デザイン比較。

## 構成

```
index.html              # 3案比較ページ
designs/
  v1-editorial.html     # 案01 / 雑誌風
  v2-noir.html          # 案02 / 黒×ゴールド
  v3-soft.html          # 案03 / やわらかウェルネス
```

## GitHub Pages へのデプロイ

1. このフォルダ内のファイルを GitHub リポジトリにアップロード
2. Settings → Pages → Source: `main` ブランチ / `/ (root)` を選択
3. 公開URLが発行されます

各案へのリンク：
- `/` または `/index.html` — 比較インデックス
- `/designs/v1-editorial.html`
- `/designs/v2-noir.html`
- `/designs/v3-soft.html`

## 補足

画像は元LPの GitHub Pages（`takeshibreks-glitch.github.io/chioma-lp/`）から外部参照しています。画像も自前でホストする場合は、元LPから画像ファイルをダウンロードし、各HTMLの画像URLを書き換えてください。
